# Topics in Quant Research

A collection of interactive explorations of ideas in quantitative finance, probability, and statistics. Not exhaustive — just things worth knowing.

Each topic is a self-contained interactive page with mathematical exposition, visualizations you can manipulate, and worked-through calculations.

**Live site:** [tsinghdev.github.io/quant](https://tsinghdev.github.io/quant/)

## Structure

The site lives in `site/` and is built with Vite + React + TypeScript. Topics are under `site/src/topics/`. Math is rendered with KaTeX.

Deployed to GitHub Pages via the `gh-pages` branch.

## Local development

```
cd site
npm install
npm run dev
```
